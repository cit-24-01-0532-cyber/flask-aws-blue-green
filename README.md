# ☁️ Multi-Environment Flask Application Deployment with Terraform & GitHub Actions

This repository demonstrates a fully automated, production-ready DevOps CI/CD pipeline that provisions a multi-environment (Blue/Green architecture concept) infrastructure on AWS using Terraform and automates application deployment via GitHub Actions.

## 🚀 Architecture & Tech Stack
* **Frontend/Backend:** Flask (Python)
* **Infrastructure as Code (IaC):** Terraform
* **Cloud Provider:** Amazon Web Services (AWS)
* **CI/CD Pipeline:** GitHub Actions
* **Deployment Model:** Elastic Beanstalk / EC2 (Multi-environment configuration)
